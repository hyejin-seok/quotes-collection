## Description

&nbsp;Welcome to the Quotes Collection application! This application allows users to explore inspirational quotes, save them to their personal collection, view detailed information about each quote, and even X(tweet) their favorite quotes. <br />
&nbsp;Built with React and Vite, and styled with Bootstrap CSS, this application provides a seamless user experience.

## Features

- **Browse Quotes**: <br />
&nbsp;Users can explore a curated collection of inspirational quotes sourced from an external API.
- **Save Quotes**: <br />
&nbsp;Users can save their favorite quotes to their personal collection for later reference.
- **View Details**: <br />
&nbsp;Detailed information about each quote, including the author and source, can be viewed within the application.
- **Tweet Quotes**: <br />
&nbsp;Users can easily share their favorite quotes on Twitter with the click of a button.

## Technologies Used
-React.js: A JavaScript library for building user interfaces.<BR />
-Vite: A fast and lightweight build tool for modern web development.<BR />
-Bootstrap: A popular CSS framework for building responsive and mobile-first websites.<BR />
-React Router: A library for declarative routing in React applications.

## Setup Instructions

1. Clone the repository to your local machine.
2. Install dependencies using npm or yarn: `npm install` or `yarn install`.
3. Start the development server: `npm start` or `yarn start`.
4. Open your browser and navigate to `http://localhost:5173` to view the application.
